# Quickstart: Arch Quiz Platform

**Created**: 2025-12-29

## Prerequisites

- Node.js 20 LTS
- MongoDB connection string
- SMTP credentials for email delivery

## Environment Variables

- MONGODB_URI
- AUTH_SECRET
- SMTP_HOST
- SMTP_PORT
- SMTP_USER
- SMTP_PASS
- APP_BASE_URL

## Local Run (outline)

1. Install dependencies.
2. Configure environment variables.
3. Start the development server.
4. Open the app in a browser and log in as an instructor.
5. Create a quiz, upload sample students and questions, and send invitations.

## Smoke Test

- Instructor can publish a quiz and send invites.
- Student can verify OTP and start a quiz.
- Instructor monitoring view shows an active attempt within 10 seconds.
